# Effective Bank

Complementary materials for the course **[Effective Coding Principles and Patterns in Java](https://devchampions.com/training/java)**. The codebase demonstrates some unique and practical coding techniques and patterns. 

### Requirements
- [x] Docker
- [x] Java 16+

#### How to run tests?
```
./gradlew test
```

#### How to run dev mode?
```
./gradlew generateJooq
./gradlew bootRun
```
